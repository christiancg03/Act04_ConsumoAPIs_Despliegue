# Detalle de Camiseta (`<CamisetaDetalle />`)

## Descripción

Este componente muestra la información completa de una camiseta de fútbol.
Obtiene el producto a partir del parámetro `id` de la URL y muestra:

- Imagen del producto
- Nombre
- Descripción
- Precio
- Categoría
- Botón para volver atrás
